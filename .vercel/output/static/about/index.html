Acerca de este proyecto

Propósito del Proyecto

Este proyecto fue creado para explorar y demostrar la funcionalidad de Monaco Editor, el poderoso editor de código que impulsa Visual Studio Code de Microsoft. Proporciona una forma de previsualizar código HTML en tiempo real utilizando las robustas capacidades de edición de Monaco Editor.
